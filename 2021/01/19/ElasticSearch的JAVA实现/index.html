ElasticSearch的JAVA实现

ElasticSearch的Java Api实现方式有四种：

  • TransportClient
  • RestClient
  • Jest
  • Spring Data Elasticsearch

因为我使用的是ElasticSearch7.6.2版本，所以使用RestClient（官方推荐）实现，官方给出来的RESTClient有Java Low Level REST Client和Java Hight Level REST Client两个，我使用Java Hight Level REST Client。
